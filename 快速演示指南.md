## 🎯 演示脚本示例

### 开场白
```
今天给大家演示 MCP 的三大核心概念：
Tools、Resources 和 Prompts

我已经准备了三个实际可运行的 demo
让我们看看它们是如何工作的
```

### Tools 演示
```
首先是 Tools - 让 AI 能够执行操作

[运行] node demo-client.js

大家可以看到，我定义了两个工具：
1. 计算器 - 执行数学运算
2. 天气查询 - 获取城市天气

当我请求"计算 123 + 456"时
Claude 自动调用了 calculate 工具
返回结果 579

这就是 Tools 的威力 - AI 不仅能思考，还能行动
```

### Resources 演示
```
接下来是 Resources - 为 AI 提供数据访问

[继续演示]

我定义了几个数据资源：
- 用户列表
- 产品目录
- 系统配置

当我请求"显示用户列表"时
Claude 读取了 data://users 资源
直接返回了所有用户信息

Resources 让 AI 能访问实时数据
就像给它打开了"图书馆"
```

### Prompts 演示
```
最后是 Prompts - 标准化工作流程

[继续演示]

我定义了四个提示模板：
- 代码审查
- 文档生成
- Bug 分析
- API 设计

以代码审查为例
我只需要提供代码和语言
模板会自动生成完整的审查提示
包含质量、性能、安全等多个维度

Prompts 将最佳实践固化为可重用模板
确保每次执行都遵循标准流程
```

### 总结
```
通过这三个 demo，我们看到：

Tools - 执行操作（改变世界）
Resources - 获取信息（感知世界）
Prompts - 标准流程（高效工作）

三者结合，就能构建强大的 AI 应用
```
